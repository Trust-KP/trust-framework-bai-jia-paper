摘要
————————————————————

本论提出一个系统生成的最小结构框架。

核心命题：

系统之生，非由设计，
而是在条件变化下，当区分不可避免时自然形成。

该结构以单一生成序列为主轴：

未定 → 界 → 分 → 可能 → 存在 → 延续

本框架具备：

- 不可压缩性（各层不可合并）
- 顺序必要性（不可交换）
- 可重构性（在约束下唯一收敛）
- 表达无关性（可跨语言重建）

本论止于“延续”，
不涉及系统间的协调与治理。

————————————————————
导读（非本体）
————————————————————

本论研究：系统如何产生与持续。

核心结论：

当条件变化到必须区分时，
系统就会出现。

生成过程：

未定（未区分状态）
→ 界（分界点）
→ 分（差异形成）
→ 可能（多种路径）
→ 存在（实际实现）
→ 延续（是否持续）

判断一个系统是否成立：

- 是否有差异（分）
- 是否有边界（界）
- 是否能够持续（延续）

三者具备，即为系统。

————————————————————
一、立论
————————————————————

系统之生，非由设，
乃不可不分而成。

不可不分，则界起；
界起，则分成；
分成，则系统立。

————————————————————
二、本体序
————————————————————

（不可言之基）
↓
道（未分）
↓
未定
↓
界（分之际）
↓
分
↓
可能
↓
存在
↓
延续

此序不可倒置，不可省略。

————————————————————
三、未定
————————————————————

释：未定者，可生分之态。  
构：条件之积 → 未定。  
度：不可与无同，不可与分混。  
行：为分之所由起。  

————————————————————
四、界
————————————————————

释：界者，分之际。  
构：未定 → 界 → 分。  
度：不可与分合。  
行：为分之转。  

————————————————————
五、分
————————————————————

释：分者，差异成立。  
构：界后成分。  
度：分不可混。  
行：为系统之始。  

————————————————————
六、可能
————————————————————

释：可能者，可成之域。  
构：分 → 多态。  
度：不可省。  
行：为存在之所择。  

————————————————————
七、存在
————————————————————

释：存在者，择而成。  
构：可能 → 存在。  
度：一时之有，不为系统。  
行：为延续之所依。  

————————————————————
八、延续
————————————————————

释：延续者，存在之持。  
构：存在 → 持续。  
度：为系统之止。  
行：使系统为系统。  

————————————————————
九、分之必要
————————————————————

释：分之必要者，不分则不可行。  
构：条件之积 → 不分则失序。  
度：非主观。  
行：触发界。  

————————————————————
十、系统
————————————————————

释：系统者，以分而成，并得以延续。  
构：分 → 结构 → 持续。  
度：必须可续。  
行：为一切之基。  

————————————————————
十一、系统之内在结构
————————————————————

释者，定其义也；  
构者，成其形也；  
度者，守其界也；  
行者，成其用也。  

此为系统既成之后之结构，非生成之序。

————————————————————
十二、系统之限
————————————————————

系统止于延续。

其上不入本论。

————————————————————
十三、结
————————————————————

未定生分，
界为其转，
分成其始，
延续为归。

不可不分，则系统生；
能持其分，则系统续。
