# Four-Pillar Analysis of the Hundred Schools of Thought

## 核心命题

诸子百家非争同一真理。

各家各执四柱之一面，以为全体。
四柱完整者，系统能久；
缺柱者，随人而亡。

> The schools of thought did not compete for the same truth.
> Each pursued one dimension of the same irreducible structure
> to its limit.
> Systems with all four dimensions survive.
> Systems with missing dimensions collapse with their founders.

---

## 四柱定义

| 柱 | Pillar | 功能 |
|---|---|---|
| 釋 | Interpretation | 定义意义，解释世界 |
| 構 | Structure | 形成稳定结构，可传承 |
| 度 | Regime | 约束边界，规范行为 |
| 行 | Operation | 执行操作，落地实践 |

**道为体，不在四柱之内——为四柱之前置条件。**

---

## 各家四柱完整性分析

### 儒家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | ✅ | 论语、孟子——仁义礼智，意义系统 |
| 構 | ✅ | 三字经、千字文、四书五经——知识结构化，可传承 |
| 度 | ✅ | 礼记、周礼、仪礼——行为规范，社会边界 |
| 行 | ✅ | 科举制度、家礼、日常礼仪——执行机制 |

**诊断：四柱皆具**

> 儒之所以久者，非仁义之美也，乃四柱皆备也。
> 论语定其释，三字经成其构，礼记守其度，科举行其用。
> 四柱皆具，故能历两千年而不衰。

---

### 道家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | ✅ | 道德经——道的意义，无为的解释 |
| 構 | △ | 庄子——结构性叙事，但非系统化知识结构 |
| 度 | △ | 无为而治——约束存在但刻意模糊 |
| 行 | △ | 修炼实践，但制度化程度低 |

**诊断：釋极强，其余三柱弱**

道家穷尽了体——触及不可分之基。
但因刻意不立構不立度，
故影响深远但难以制度化。

> 道家近道，故最深；
> 然无构无度，故难以为治。

---

### 法家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | △ | 有理论基础但以实用为主 |
| 構 | △ | 法典结构存在但非知识传承体系 |
| 度 | ✅ | 法、术、势——约束机制极强 |
| 行 | ✅ | 赏罚执行，制度落地 |

**诊断：度行极强，釋構偏弱**

法家能成一时之治，
但缺乏意义系统和知识传承结构，
故秦用之而速亡，
汉后以儒补其釋構之缺。

> 法有度有行，然无释无构，
> 故能立法于一时，不能化民于万世。

---

### 墨家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | △ | 兼爱非攻——有意义主张但系统性不足 |
| 構 | △ | 墨经有逻辑结构，但传承体系弱 |
| 度 | △ | 有纪律约束，但制度化程度低 |
| 行 | ✅ | 节用、实践、工程技术——执行极强 |

**诊断：行强，三柱皆弱**

墨家最重实践，
然缺乏稳定的知识传承结构，
随创始人之后逐渐式微。

> 墨重行而轻构，
> 故技艺精而传承断。

---

### 纵横家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | ✅ | 解读形势，识别各国利益，游说之术 |
| 構 | ✘ | 无稳定知识结构，无著书传承 |
| 度 | ✘ | 无约束机制，联盟无制度支撑 |
| 行 | ✅ | 游说执行，实际操作联盟 |

**诊断：釋行具备，構度皆无**

有釋有行，故能成于一时；
无構无度，故随人而亡。

苏秦死，合纵即解；
张仪走，连横即变。

> 纵横有释有行，无构无度，
> 故能成于一时，不能立于万世。

---

### 名家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | ✅ | 名实之辨，概念精确性极强 |
| 構 | △ | 有逻辑结构，但体系不完整 |
| 度 | ✘ | 无约束机制 |
| 行 | ✘ | 无实践执行系统 |

**诊断：釋极强，度行皆无**

名家将釋推到极致——
白马非马，坚白石之辩。
然无度无行，
故精致而无用，影响有限。

> 名家尽其释，然无以行，
> 故辩才无碍，不能经世。

---

### 阴阳家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | ✅ | 宇宙生成解释，阴阳五行意义系统 |
| 構 | ✅ | 五行相生相克，结构系统完整 |
| 度 | △ | 有约束逻辑但制度化弱 |
| 行 | △ | 有实践但缺系统执行机制 |

**诊断：釋構强，度行弱**

阴阳家有完整的宇宙结构，
后被儒家吸收整合，
成为儒家構层的一部分。

> 阴阳有释有构，
> 故被吸收而不消亡，
> 乃构入儒学，以延其续。

---

### 兵家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | ✅ | 形势判断，战略解读 |
| 構 | △ | 有兵法结构，著书传承 |
| 度 | ✅ | 军纪约束，赏罚分明 |
| 行 | ✅ | 战术执行，实战操作 |

**诊断：釋度行强，構偏弱**

兵家近于完整，
故孙子兵法能传两千年。
然兵家之構限于军事领域，
不能扩展为文明级系统。

---

### 农家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | △ | 重农思想，但意义系统不完整 |
| 構 | △ | 农业知识结构，但传承体系弱 |
| 度 | △ | 有生产规范，但制度化弱 |
| 行 | ✅ | 耕作实践，生产操作 |

**诊断：行强，三柱皆弱**

农家重实践轻理论，
故技术有用但思想影响有限。

---

### 杂家

| 柱 | 状态 | 载体 |
|---|---|---|
| 釋 | △ | 兼采各家 |
| 構 | △ | 兼采各家 |
| 度 | △ | 兼采各家 |
| 行 | △ | 兼采各家 |

**诊断：无独立柱，皆为组合**

杂家无法独立成柱，
只能混合已有四柱。
这本身证明了四柱的完整性——
无第五柱可立。

> 杂家兼采，无独立之柱，
> 此四柱充分之旁证也。

## 核心模式

**四柱完整 → 文明级延续（儒家）**

**釋構强，度行弱 → 影响深远但难制度化（道家、阴阳家）**

**度行强，釋構弱 → 一时之治，难以持久（法家）**

**只有釋行，无構无度 → 随人而亡（纵横家）**

**只有行，无其他 → 技术传承，思想影响弱（墨家、农家）**

**无独立柱 → 无法独立传承（杂家）**

## 总纲

> 道为体，易构其形，儒释其义，法度其界，墨行其用。
>
> 四柱皆具者久，缺柱者亡。
>
> 百家非争，乃分执一面；
> 执一面者，各有其极；
> 四面皆执者，方能经世致用，历久而不衰。
>
> 此非诠释，乃收敛。
> 此非发明，乃发现。
