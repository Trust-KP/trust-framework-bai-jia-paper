# Theory of Meta System Architecture

This repository establishes the original conceptual formulation of the
Theory of Meta System Architecture.

**Author:** Kok Pian Gew
**Status:** Public record
**Date:** 2026

---

## Overview

The Theory of Meta System Architecture is a foundational ontological
framework for examining how any system — across any domain — comes to
exist, becomes observable, and persists across time.

The theory proposes that all systems arise through a single
ontological progression, and that once a system exists, it can be
completely described through four irreducible observational dimensions.

Core ontological path:

> Dao — Indeterminacy — Boundary — Distinction — Possibility —
> Existence — Continuation

Core observational dimensions:

> Interpretation · Structure · Regime · Operation

A system enters existence as a recognisable entity at the moment
distinction is made and records begin. What is not recorded remains
hidden. What is recorded becomes visible.

The theory is complete when its own four dimensions are fully present
within itself.

---

## The KP Conjecture

Any system, across any domain, is fully expressible through the
six-state ontological path from Indeterminacy to Continuation, and
through the four irreducible dimensions of Interpretation, Structure,
Regime, and Operation.

Four properties follow:

- **Sufficiency** — The four dimensions are sufficient to describe
  any system completely. No examined system has required a fifth.
- **State invariance** — The six-state path is fixed in sequence
  and cannot be compressed or reordered.
- **Path invariance** — Any of the four dimensions may serve as the
  entry point. The underlying path remains unchanged.
- **Recursive completeness** — Every subsystem is itself a complete
  system. There are no partial systems.

This is not a theorem. It is a constraint. To refute it is to produce
a system that cannot be described within it.

## Relationship to Other Works

This theory establishes the ontological foundation from which the
Trust Framework proceeds.

Where this theory ends — at continuation — the Trust Framework begins.

If the ground of all systems is what precedes distinction, then
continuation is what follows it — and everything between is system.
