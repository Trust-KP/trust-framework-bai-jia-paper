————————————————————
【0. 定义】
————————————————————
三字经不是儿童读物。

三字经 = 个体校准架构（Individual Calibration Architecture）

其功能在于：
将系统规则（度）内化为稳定行为（行），
确保个体在不确定条件下维持可延续的决策路径。

————————————————————
【1. 本体假设层（Human Ontology）】
————————————————————
人之初 性本善

定义：
- 个体初始状态为可合作（baseline cooperative potential）

功能：
- 提供行为系统的起点假设
- 降低系统复杂性（默认信任起点）

对应框架：
本体 + 释（interpretive prior）

————————————————————
【2. 分化机制层（Differentiation）】
————————————————————
性相近 习相远

定义：
- 个体差异来自环境与交互

功能：
- 引入路径依赖（path dependency）
- 解释行为分布差异

对应框架：
分 → 构

————————————————————
【3. 期体形成层（Expectation Formation）】
————————————————————

定义：
- 期体由环境与教育塑造

功能：
- 建立未来行为预期模型
- 决定决策基线

对应框架：
期体（expectation layer）

————————————————————
【4. 体制内化层（Regime Internalisation）】
————————————————————

- 将外部“度”转为内部约束
- 防止行为漂移

对应框架：

————————————————————
【12. 系统性质】
————————————————————
1. 内化性：
- 外部规则转为内部驱动

2. 路径依赖：
- 早期状态影响长期结果

3. 可塑性：
- 可通过环境调整改变路径

4. 闭环性：
- 行为 → 反馈 → 更新

5. 延续敏感性：
- 初期偏差可导致长期失效

————————————————————
【13. 失效模式】
————————————————————
道错（释错）：
- 错误理解“何为善”

象错（构错）：
- 错误环境 / 结构

理错（度错）：
- 规则不稳定 / 不一致

行错（行错）：
- 不执行 / 执行
